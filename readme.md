# How to use


1. ``` git clone https://github.com/imnkywf/crime_backend.git```

2. ``` cd crime_backend ```

3. ``` python3 -m venv venv  ```

4. ``` cd venv ```

5. ``` source bin/active ```

6. ``` cd crime_backend ```

7. ``` pip3 install -r requirements.txt ```

8. ``` python3 manage.py runserver ```
